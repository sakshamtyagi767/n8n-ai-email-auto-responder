# AI Email Auto Responder (n8n)

## 🚀 Overview
This project automates email handling using n8n and AI.

It:
- Reads incoming emails
- Filters unwanted messages
- Generates smart replies
- Sends replies automatically

---

## 🛠️ Tech Stack
- n8n
- Gmail API
- OpenAI API

---

## ⚙️ Workflow
Gmail Trigger → IF → SplitInBatches → Wait → AI → Gmail Send

---

## 📦 Setup
1. Import workflow.json into n8n
2. Add credentials
3. Activate workflow
